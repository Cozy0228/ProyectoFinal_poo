Proyecto Final Asignatura Programacion Orientada a Objetos
-------------------------------------------------------------------------------------------------------------------
Bienvenido!
El equipo de trabajo Unidentified Programmable Object (UPO) te de la bienvenida a este repo en el cual se evidenciara y dara a conocer el desarrollo de un problema como proyecto final semestral de la materia Porgramacion Orientada a Objetos (POO).

El problema que se escogio para trabajar fue de un *sistema de gestion de inventario para una bodega* que sera realizado por medio del lenguaje de programacion PYTHON las especificaciones que se llevaran a cabo son las siguientes:
- Codigo original
- Uso de herramientas vistas en el curso (CLASES)
- Creacion de objetos a almacenar, +20 con diferentes atrubitos el principal seria precio.
- Operaciones de registro de entrada y salida.
- Un metodo para obtener un listado de inventario actual.
- Manejo de fechas en los registros.

Dichas indicaciones se iran trabajando y documentando en el repo con mas detalle para el usuario.
